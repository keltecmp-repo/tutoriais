PASSO 01 / 05
Ativar Fontes Desconhecidas
01

O Kodi bloqueia instalações de fontes não oficiais por padrão. É necessário liberar esta opção antes de prosseguir.

1
Na tela inicial, clique no ícone ⚙️ ENGRENAGEM (Configurações)
2
Selecione SISTEMA
3
Vá em ADD-ONS
4
Ative a opção "FONTES DESCONHECIDAS"
⚠️
Atenção Ao ativar, clique em SIM para confirmar. Instale apenas de fontes confiáveis como o KelTec Media'Play!
📁
PASSO 02 / 05
Adicionar Fonte no Gestor de Arquivos
02

Adicione o repositório oficial KelTec Media'Play como fonte de arquivos no Kodi.

1
Volte para a tela inicial
2
Clique em ⚙️ CONFIGURAÇÕES
3
Selecione GESTOR DE ARQUIVOS
4
Clique em "ADICIONAR FONTE"
5
Digite ou cole o endereço do repositório oficial:
URL https://keltecmp-repo.github.io/repo/
6
Dê o nome KelTec Repo para a fonte
7
Clique em OK para confirmar
📦
PASSO 03 / 05
Instalar o Repositório via ZIP
03

Com a fonte configurada, agora instale o repositório KelTec Media'Play a partir do arquivo ZIP.

1
Volte para a tela inicial
2
Clique em ADD-ONS (ícone de quebra-cabeça 🧩)
3
Clique no ícone de 📦 CAIXA ABERTA (no topo da tela)
4
Selecione "INSTALAR DO ARQUIVO ZIP"
5
Escolha a fonte: KelTec Repo
6
Selecione o arquivo ZIP do repositório:
🗜️
repository.keltecmp-1.0.0.zip
Arquivo do repositório oficial KelTec Media'Play
7
Aguarde a notificação de confirmação na tela
✅
Repositório Instalado com Sucesso!
O KelTec Media'Play Repository está pronto para uso
🎬
PASSO 04 / 05
Instalar o KelTec Open Wizard
04

O KelTec Open Wizard é o gerenciador oficial de Builds do KelTec Media'Play.

1
No menu ADD-ONS, clique em "INSTALAR DO REPOSITÓRIO"
2
Selecione: KelTec Media'Play Repository
3
Escolha: 🎬 ADD-ONS DE PROGRAMAS
4
Selecione: ⭐ KelTec Open Wizard
5
Clique em INSTALAR
6
Aguarde a instalação completa antes de prosseguir
🚀
PASSO 05 / 05
Instalar a Build
05

Última etapa! Instale a Build KelTec Media'Play e prepare-se para uma experiência cinematográfica completa.

1
Abra o KelTec Open Wizard
2
Escolha o menu BUILD
3
Escolha a Build desejada
4
Clique em INSTALAR e confirme no menu seguinte
5
Aguarde o download e instalação completos
6
Confirme forçar fechamento do Kodi quando solicitado
7
Abra o Kodi novamente e aproveite! 🎉
🎬
Instalação Concluída!

A Build KelTec Media'Play está instalada e pronta para transformar sua experiência de entretenimento. Abra o Kodi e curta!
